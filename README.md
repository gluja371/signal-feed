# Signal Feed

A single-user, anti-doomscroll PWA: a finite daily feed of AI learning cards
(computer vision, vibe coding, agentic AI, AI news) generated by a Claude API
pipeline. Instagram feel, newsletter soul.

## Structure

```
index.html      PWA shell (React UMD + Babel standalone, no build step)
app.jsx         The feed app — renders cards from feed.json
feed.json       Today's cards (regenerated daily by the pipeline)
manifest.json   PWA manifest
sw.js           Service worker: offline shell, network-first feed
icon-*.png      App icons
pipeline/       Content generation (Python + Claude API)
.github/workflows/daily-feed.yml   Daily cron via GitHub Actions
```

## Deploy (once, ~10 minutes)

1. Create a GitHub repo and push this folder to `main`.
2. Repo → Settings → Pages → Source: "Deploy from a branch" → `main` / root.
3. Repo → Settings → Secrets and variables → Actions → New repository secret:
   `ANTHROPIC_API_KEY` = your key from console.anthropic.com.
4. Actions tab → "Daily feed generation" → Run workflow (manual test).
5. Open `https://<username>.github.io/<repo>/` on your iPhone in Safari →
   Share → **Add to Home Screen**. Done: fullscreen app, no browser chrome.

## Daily loop

GitHub Actions runs `pipeline/pipeline.py` every morning: pulls the RSS lanes,
asks Claude to turn the best items into cards (see
`pipeline/generation-prompt.md`), commits a fresh `feed.json`. The app is
static — it just reads that file. Feed is capped at 8 cards/day on purpose.

## Local pipeline test

```
cd pipeline
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...
python pipeline.py
```

## Notes

- RSS URLs in `pipeline.py` LANES are a starting set — a couple are best-guess;
  verify on first run and swap freely.
- Costs: hosting/cron are free; Claude API ≈ NZ$5–15/month at 8 cards/day.
- Saved cards are session-only for now (in-memory). A `saved.json` +
  localStorage upgrade is a good first Claude Code task — localStorage works
  fine in the deployed PWA (the no-localStorage rule only applies to
  claude.ai artifact previews).
